# ユーザー向けドキュメント (USER_DOC)

## サービス概要
本プロジェクト "Inception" は、Docker コンテナを用いて構築された Web インフラストラクチャです。
以下の主要サービスで構成されています：
- **Nginx**: SSL/TLS に対応した Web サーバー
- **WordPress**: PHP で動作するコンテンツ管理システム
- **MariaDB**: WordPress のデータを保存するデータベース

## 起動・停止方法
プロジェクトのルートディレクトリで以下のコマンドを使用します。

- **起動**:
  ```bash
  make
  ```
  これにより、コンテナのビルドとバックグラウンドでの起動が行われます。

- **停止**:
  ```bash
  make down
  ```

## サイト・管理画面へのアクセス
ブラウザで以下の URL にアクセスしてください。自己署名証明書を使用しているため、セキュリティ警告が表示される場合がありますが、例外を追加して進んでください。

- **Web サイト**: https://kosakats.42.fr
- **管理画面**: https://kosakats.42.fr/wp-admin

## 認証情報管理
WordPress の管理者アカウントやデータベース接続情報は、環境変数（`.env` ファイル）によって管理されています。セキュリティのため、これらの情報はリポジトリには含めず、デプロイ時に設定されます。

## 動作確認方法
1. `docker ps` コマンドを実行し、`nginx`, `wordpress`, `mariadb` のコンテナが `Up` 状態であることを確認してください。
2. ブラウザで `https://kosakats.42.fr` にアクセスし、WordPress の画面が正しく表示されれば動作しています。
